Task Switching Cognitive Test

Instructions

This test measures your cognitive flexibility and attention control.

Task Rules:

Color Task:

Press B for YELLOW objects

Press N for BLUE objects

Shape Task:

Press B for CIRCLES

Press N for RECTANGLES

Important:

  • You will see a cue before each trial indicating which task to perform
  • Respond as quickly and accurately as possible
  • You have 2 seconds to respond to each stimulus
  • There will be a training phase followed by the actual test

Training Phase

Practice the tasks. You will complete 10 trials.

Test Phase
